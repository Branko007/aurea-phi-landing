# Aurea Phi — Landing

Landing page de **Aurea Phi**, empresa de soluciones de IA, datos, software y agentes.

Estética **geométrica minimal** sobre fondo oscuro, con un acento índigo→cian y el concepto del **número áureo (φ)** recorriendo el sitio: la espiral áurea en el hero, las proporciones del layout y las marcas geométricas de las secciones.

## Stack

Sitio estático, sin dependencias ni build:

- `index.html` — estructura y contenido
- `styles.css` — estilos
- `script.js` — nav sticky, menú móvil, scroll-reveal, scrollspy y carrusel

Tipografías (Google Fonts): **Inter** (cuerpo) y **Space Grotesk** (wordmark / nombres).

## Cómo verlo localmente

```bash
python3 -m http.server 8123
# abrir http://127.0.0.1:8123
```

## Secciones

1. Hero — espiral áurea con los servicios anclados en sus puntos φ
2. Servicios — IA, Datos, Software, Agentes
3. Cómo trabajamos — proceso de 4 pasos
4. Herramientas — carrusel de tecnologías del stack
5. Contacto / CTA
